Rule-Based Chatbot:
A basic chatbot built using rule-based logic that responds to user inputs using predefined patterns and replies. It can handle common queries and simple conversations without using machine learning,
mainly relying on if-else rules and keyword matching.

Tic-Tac-Toe Game:
A GUI Tic-Tac-Toe game where a user plays against a computer opponent. It includes game logic, move validation, and an AI player that makes decisions based on strategy instead of random moves,
along with a simple interactive interface.

Recommendation System:
A small recommendation system that suggests Indian movies to users based on their selected genre. It uses a simple preference-matching approach and a Tkinter interface to let users choose one option 
and receive at least five relevant suggestions.
